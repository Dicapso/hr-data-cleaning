# HR Data Cleaning & Dashboard

## Layihə haqqında
IBM HR Analytics datasetinin (1470 sətir, 35 sütun) təmizlənməsi 
və vizuallaşdırılması.

## Nə edildi
- Dublikat sətirlər silindi
- Boş xanalar dolduruldu
- Lazımsız sütunlar çıxarıldı
- Mətn standartlaşdırıldı

## Dashboard
- Şöbə üzrə işçi sayı
- Aylıq maaş paylanması
- Yaş paylanması
- İşdən çıxma nisbəti

## Texnologiyalar
- Python
- pandas
- NumPy
- matplotlib
